# Changelog

## Version 1.1 - December 2025

### Changed

- **Initial Stacks**: Changed from "one stack per player" to always starting with 3 stacks, regardless of player count.
  - This provides a more consistent and strategic starting state across all game modes.
  - Improves solitaire mode by providing more decision space from the start.
  - Simplifies the rule: "Always start with 3 stacks" is easier to remember than "one per player."

## Version 1.0 - November 2025

### Initial Implementation

- Core game mechanics
- Stack structure and capacity rules
- Turn rules and legal plays
- Collection conditions (Six-Six, Seven-Seven, Six-Seven, Zero, Match)
- End game conditions
- Scoring system
- Multiplayer and solitaire modes
